Pour cela, on va faire une condition if avec une expression/formule utilisant l'op√©rateur **modulo** `%`.

:::{admonition} L'op√©rateur modulo %
:class: info

L'op√©rateur modulo est un op√©rateur math√©matique au m√™me titre que `+` `-` `/` `*`
Il permet d'obtenir le reste d'une division.

Par exemple:
* `13 % 5 == 3` car 13 divis√© par `5` fait `2`, et il reste `3`.
* `523 % 34 == 13` car 523 divis√© par `34` fait `15`, puis il reste `13`. (sur ma vie le calcul est bon)

Du coup, sachant qu'un nombre **pair** est un **multiple de 2** (si on le divise par `2`, le reste de la division sera `0`), alors on peut utiliser le modulo pour savoir si ce nombre est **pair** ou **impair**.

Pour r√©sumer, on peut donc admettre que la condition `nombre % 2 == 0` sera vraie si nombre est **pair** et `nombre % 2 == 1` sera vraie s'il est **impair**.
:::
